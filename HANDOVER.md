
### Quick Deploy to Netlify:

```bash

# Option 2: Connect GitHub repo
```

## 🔧 Updating Content

All content is in **one file**: `src/pages/index.astro`

To update copy:
1. Edit the HTML content directly
2. Run `npm run dev` to preview (http://localhost:4321)
3. Run `npm run build` to rebuild
4. Deploy updated `dist/` folder

**No build needed for simple text changes if deploying via Git push** (Vercel/Netlify auto-rebuild on push).
